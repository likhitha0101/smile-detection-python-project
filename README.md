# Smile Detection Python Project

## Overview
This project detects human faces and smiles in real-time using Python and OpenCV. It uses a webcam to capture video and identifies faces using Haar Cascade classifiers.

## Features
- Real-time face detection
- Smile detection within detected faces
- Uses OpenCV Haar Cascade Classifiers
- Simple and easy-to-understand Python code

## Technologies Used
- Python
- OpenCV
- Haar Cascade Classifier

## Project Files
- smile_detection.py : Main Python program for smile detection
- haarcascade_frontalface_default.xml : Face detection classifier
